#  Bike Store Sales Dashboard

## Project Overview

This project analyzes three years of bike store sales data using SQL and Excel. The goal is to uncover business insights related to sales performance, store efficiency, employee contribution, product demand, and brand performance.

SQL was used for data extraction, transformation, and analysis. Multiple SQL views were created and imported into Excel to build an interactive dashboard.

---

## Tools Used

- SQL
- Microsoft Excel
- Data Visualization

---

## Project Workflow

1. Data Cleaning and Preparation
2. SQL Query Writing
3. Creation of SQL Views
4. Importing Views into Excel
5. Dashboard Development
6. Business Insight Generation

## Dashboard KPIs

- Total Sales: $7.69M
- Total Orders: 4,722
- Average Order Value: $4,761
- Total Discount: $0.89M
- Number of Stores: 3
- Number of Employees: 10
- Number of Brands: 9

- Top 5 Selling Products

### Employee Performance

### Brand Analysis

### Regional Analysis
- State-wise Sales Comparison

---

## Key Insights

- Baldwin Bikes generated the highest revenue.
- Trek is the top-performing brand.
- Electric Bikes contribute the highest category sales.
- New York is the strongest sales region.
- Top employees contribute a significant share of total revenue.
- Discounts account for approximately 11.5% of total sales.

---

## Business Recommendations

- Increase focus on high-performing brands and products.
- Improve sales performance in lower-performing stores.
- Invest more in Electric Bike promotions.
- Optimize discount strategies.
- Replicate successful sales practices across teams.
